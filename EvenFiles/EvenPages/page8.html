Recap: the Head

As we have learned, every web page has a head and body. The head is where the pageâ€™s metadata is found, along with everything else that is meant not to appear on the visible part of the page.

Five types of things are found in the head:
  • Title: the title is what shows up at the top of the web page window or tab.
  • Metadata: data that is invisible to the viewer, but is visible to web crawlers, and thus important when the page appears in search results.
  • Styles: instructions to the web browser on how the web page should look.
  • Scripts: JavaScript code that must run before the body of the web page loads.
  • Links: important files that are linked to the page, including external CSS and JavaScript files.
Example: 
    <head>
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width">
      <title>Step 1 on how to bake cookies</title>
      <meta name="author" content="[Yukti Solanki]">
    </head>
