Headings & Paragraphs

(Lesson 3)

Headings;

Headings are used to create an outline structure on a web page.There are 6 heading types: <h1> through <h6>. Aside from the size of the text, headings allow assistive devices like screen readers. All headings are block-level elements. 

The 6 heading types;

    <h1>The top-level heading </h1> 
    <h2> Second level </h2> 
    <h3> third level </h3>
    <h4> fourth level </h4> 
    <h5> fifth level </h5>
    <h6> Sixth level </h6>
Paragraphs:

Paragraph <p> is the workhouse tag of web design. Most blocks of text use paragraph for their formatting. Paragraphs are block-level elements, creating space above and below them on the page. (If you want to create a line break, use the <br> tag) 

    <p>Most text in the web pages are formatted in paragraphs.</p> 
    <p> The line break tag <br> 
    will end the text at the point it is inserted <br> 
  and continue on the next line. <br> 
  is an inline tag.</p>
Horizontal Rule and Preformatted text:

If you want to create a line between blocks of text, use the <hr> element. Hr is an empty tag. If you want to preserve multiple spaces or line breaks for text like in poems, you can use the preformatted text <pre> tag. Pre is a block-level element.

Example:

    <p>Poem</p>
    <hr

    <pre>
  The text in this 
    Tag will 	display 
  Exactly as 

    You see it 
    </pre>
(Extra) Loren Ipsum Text

Lorem Ipsum text was created long before the use of computers so that book designers could format their products before the writers delivered the content. Web designers use it now to mock up pages.
