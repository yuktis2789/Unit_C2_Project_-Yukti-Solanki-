Understanding HTML

(Lesson 1)

In HTML, the markups are called tags, and are shown with the < and > symbols, which we call angle brackets.

Writing HTML code (HTML code can be written in any TextEditing program):

  • On Windows, you can edit HTML using the NotePad program.
  • On the Mac, you can edit HTML using the TextEdit program, configured for plain text editing.

Essential HTML elements:

A properly written web page should have the following elements:

  1. The HTML5 doctype declaration.
  2. The html tag begins and ends the page.
  3. The head tag holds the pageâ€™s metadata.
  4. The title tag gives the words that show up on web browser tabs.
  5. The body tag contains the visible content of the page.

Example:

    <!DOCTYPE html>	
    <html>
    <head>
      <title>Title of Web page </title>
     <head>

     <body>
     <h1>Main heading of Web Page </h1>

     <p>Paragraph content uses the p tag. </p>
     </body>
