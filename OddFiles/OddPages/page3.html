Formatting Tags and Elements

(Lesson 2)

Types of tags: HTML has two types of body elements: 
    Block-level elements will automatically appear on separate 
lines on the page.
    Example: <p> and <h1> 
    Inline elements can appear next to one another horizontally, 
so that text with an inline element in it will not start 
a new line. 
    Example: <strong> and <i>

Formatting Tags: (Several elements modify the look of text within them.)

These are all inline elements: (you can use them to modify one or more characters within a line of text).
  • <b>This will appear bold </b>
  • <strong>This will appear bold </strong>
  • <i>This will appear in italics </i>
  • <em>This will appear in italics </em>
  • <mark>This will appear highlighted </mark>
  • <small>This text will be smaller </small>
  • <del>This will appear crossed out </del>
  • <ins>This will appear underlined </ins>
  • <sub>This will appear as subscript </sub>
  • <sup>This will appear as superscript </sup>
List Elements: There are two types of list tags: 
      <ol> creates an ordered list (a list with numbers). 
      <ul> creates an unordered list (a list with bullets
      instead of numbers).

Both list types use <li> (list item) for the items within the lists. ol, ul and li are block level elements

 
Example:
    
    <ol> 
      <li>list</li> 
  <li>is</li> 
      <li>numbered</li> 
      <ol> 

    <ul> 
      <li> list</li> 
      <li> has</li> 
      <li> bullets</li> 
      </ul>
