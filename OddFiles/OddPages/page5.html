Comments & Attributes

(Lesson 4)

Comments:

Comments are text hidden within an HTML document. In order to create an HTML comment, you must use the characters “<!--” before the comment text and “-->” after it.

Example: 
  <!-- This is a single line comment -->
  <!-- This is 
  a multiple line
  Comment -->
  <!-- You can disable elements 
  <h1>Hello World</h1> by enclosing them in a comment -->
Attributes

Attributes are codes that appear within a tag that modify it in some way. The general syntax for an attribute is:

<tagName attributeName=”value”>content </tagName>
Example: 
  <img src=”happyface.jpg”>
  <p style=“color:red”>This paragraph will have red text</p>
Links:

Links use the anchor or <a> element to make a hyperlink to another page or to another location on the same page. It is an inline element. To locate the link, you need the href attribute. The general syntax for a link is:

<a href=”pathToFile”LinkText</a>
Example: 
  Absolute or external link: 
  <a href=”https://google.com”>Google</a>
    
  Relative or internal link:
    
  <a href=”/index. html”>HomePage</a>
More link attributes:

Download: let’s the user download a file.
Target: specifies a specific place within the page the link opens.
Some targets:

  • _blank: opens in a new window/tab
  • _top: opens at the top of the page
  • _self: opens within the same iframe
  • _parent: opens within the parent of the current frame
Example 
    <a download=“fileName”>...
      <a href=”URL” target=”_blank”>...
Embedding Images

Images are embedded on a web page using the <img> element. It is an empty element (no trailing tag) and works inline. To locate the image, you must use the src attribute. You must also use the alt attribute to describe the image for screen readers or if the image will not load.

Example 
    Local image: 
    <img src=”happyface.jpg”
    alt=”Smiley Face”>

    Remote image: 

    <img src=”https://ex.com/hf.jpg”
    alt=”Smiley Face”>
Sizing images

You can control the size of images by using either width or height attributes, or both. The actual dimensions are specified in pixels. If you only specify the height or width, the image with scale proportionally. If you specify both, the image will fix itself to the dimensions you chose.

Example: 
<img src=”file” 
width=”200”>
