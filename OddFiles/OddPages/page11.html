The div tag

HTML4 introduced the <div> tag which allowed web developers ot create divisions, or sections to their pages. Web pages soon had many divs containing different types of content. While useful, overuse of div elements can make it difficult to understand how the page is structured. The div tag is also a non-semantic element.

Other semantic tags 
<h1> - <h6> These tags mean the heading of the webpages in 
number descending order. 
<a> this tag means it is a hyperlink. 
<button> this tag name means a button element.
The benefits of semantic HTML tags

The most important benefit of using semantic HTML tags would be the clearness that the code has because of it. There is a better understanding of the code since there is now a purpose and it is also well structured and organized so it behaves greatly as a semantic HTML tag. It is able to be more search engine friendly and much more readable to whatever and whoever is reading the code with semantic HTML tag.
